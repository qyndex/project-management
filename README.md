# Project Management

Kanban board with drag-and-drop, calendar view, team management, and notifications.

**Stack:** Vite | **Complexity:** complex
**Tags:** `kanban`, `project-management`, `calendar`, `drag-and-drop`, `dnd-kit`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
